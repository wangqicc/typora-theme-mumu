## 使用方法

1. 打开Typora，依次选择`文件` -> `偏好设置` -> `外观` -> `打开主题文件夹`

2. 将`wood.css`和`wood-dark.css`复制到文件夹中

3. 依次选择`主题` -> `Wood`或`Wood Dark`即可
